Prework Study Guide

HTML

  • Stands for Hypertext Markup Language.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Character set as UTF-8 to allow us to use most characters and symbols
  • P element in the body represents a paragraph or block of context.

CSS

  • Stands for Cascading Style Sheets
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS: Used to style an element directly in the HTML file. Used for quick and permanent changes but must be edited individually when making a design change.
  • Interal Style Sheet: Contains CSS rules for the webpage in the head section of the HTML file.
  • External CSS Style Sheet: Allows to keep all CSS rules in a separate file which makes design changes easier. Use link element to link a file.
